# IdeaBox

IdeaBox 是一款以 SwiftUI 打造的 iOS 應用程式，提供快速、簡單的靈感紀錄體驗。專案遵循正體中文文件與測試優先原則，確保每次迭代都具備穩定品質與清晰結構。
